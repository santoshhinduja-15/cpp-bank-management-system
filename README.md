# C++ Bank Management System
A console-based Bank Management System developed in C++ using Object-Oriented Programming and text file handling. The application simulates basic banking operations with proper validation and persistent data storage.

## Features
- Create new bank accounts
- Deposit and withdraw money with validation
- Prevent zero or negative transactions
- Check account balance
- Display all account details
- Persistent storage using text files
- Menu-driven console interface

## Technologies Used
- C++
- Object-Oriented Programming (OOP)
- File Handling (Text Files)

## Input Validation
- Deposit or withdrawal amounts must be greater than zero
- Negative or zero transactions are rejected with an error message
- Withdrawals exceeding available balance are not allowed

## Future Enhancements
- Add PIN-based authentication
- Replace file storage with a database
